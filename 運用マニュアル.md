# andtete HP 運用マニュアル

## 無料で維持する構成

- HP公開: GitHub Pages、Netlify、Cloudflare Pagesなどの無料枠
- 独自ドメイン: 使わない
- 決済: Stripe Payment Links
- 顧客管理: Stripe管理画面
- お問い合わせ: 公式LINE

Stripeは月額費用なしで使えますが、決済が発生したときの決済手数料はかかります。

## 商品追加・削除の流れ

1. `admin.html` を開く
2. 商品名、価格、在庫数、説明、Stripe購入リンクを入力する
3. 商品写真を最大7枚まで選ぶ
4. 掲載したいカテゴリにチェックを入れる
5. 「この商品を保存」を押す
6. 「HPで確認」で表示を確認する
7. 「公開用ファイルを作る」を押す
8. ダウンロードされた `products.json` を `data/products.json` と差し替える
9. ダブルクリックで確認したい場合は、同じ内容を `data/products.js` にも反映する

通常の公開サイトでは `data/products.json` を読み込みます。ファイルを直接開く環境ではブラウザ制限でJSONを読めないことがあるため、予備データとして `data/products.js` も入れています。

## 商品カテゴリ

- `NEW ARRIVALS`: 新商品欄に表示
- `おすすめ`: おすすめ欄に表示
- `予約商品`: トップと予約ページに表示
- `即納商品`: 即納商品欄に表示

1つの商品を複数カテゴリに掲載できます。

トップページでは `NEW ARRIVALS` と `おすすめ` は横スライド表示に加えて、「一覧で見る」から大きめのカード一覧でも確認できます。

## 在庫数

商品管理画面の「在庫数」に数字を入れると、HP上で `残り○点` と表示されます。
在庫数が `0` の商品は `在庫なし` と表示され、購入ボタンも押せない表示になります。

過去のテスト商品や古い保存内容を消したい場合は、商品管理画面の「初期データに戻す」を押してください。

## Stripeとのつなぎ方

1. Stripe管理画面で商品を作る
2. Payment Linksで購入リンクを作る
3. 発行された `https://buy.stripe.com/...` のURLを商品データの「Stripe購入リンク」に貼る
4. HP上の「購入する」ボタンからStripe決済ページへ移動できる

Payment LinksはStripe公式ドキュメントでも、コードを書かずに商品販売用の決済リンクを作れる方法として案内されています。

## 決済後の顧客管理

決済が完了した顧客情報、支払い履歴、返金対応はStripe管理画面で確認します。
HP側に顧客情報を保存しないため、無料の静的サイトとして維持できます。

## Googleスプレッドシート連携

1. Googleスプレッドシートを新規作成する
2. メニューの「拡張機能」から「Apps Script」を開く
3. `google-apps-script.gs` の内容を貼り付ける
4. `setupShare` を1回実行して、`and.tete03@gmail.com` に編集権限を付与する
5. 「デプロイ」から「ウェブアプリ」として公開する
6. アクセスできるユーザーは「全員」にする
7. 発行されたURLを管理画面の「スプシ連携URL」に貼る
8. 「連携URLを保存」を押す
9. 商品を保存したあと「スプシへ送信」を押す

送信される列は、商品ID、商品名、価格、在庫数、ステータス、掲載カテゴリ、HP表示、Stripeリンク、説明、写真枚数、更新日時です。

Google側の権限付与はGoogleアカウントの認証が必要なため、スプレッドシート所有者がApps Script上で `setupShare` を実行してください。

この連携は「管理画面の商品リストをスプシへ送る」仕組みです。スプシ側で在庫を変更した内容をHPへ自動で戻すには、追加のGoogle認証または公開CSV設定が必要です。

## 画像を登録する方法

1. `admin.html` を開く
2. 「商品写真」で画像ファイルを選ぶ
3. 最大7枚までまとめて登録できる
4. 自動で正方形に切り抜き・リサイズされる
5. 1枚目がメイン写真になる
6. サムネイルを押すとメイン写真を入れ替えられる
7. 商品を保存する
8. 「HPで確認」で見え方を確認する

画像サイズは商品一覧に合うように自動で統一されます。

## 公開前に必要な作業

- ロゴ画像がある場合は差し替える
- 実際の商品写真に差し替える
- Stripe Payment Linksを作成して商品に貼る
- GitHub Pagesなどの無料ホスティングへアップロードする
- SNS、LINE、TikTokリンクが正しいか確認する
